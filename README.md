# Atesh Grill Bremen 🌯🔥

Willkommen im GitHub-Repository der Website von **Atesh Grill Bremen**.

Diese Website dient als digitale Präsenz unseres Restaurants und bietet Gästen einen einfachen Überblick über unser Angebot, unsere Speisen und Kontaktmöglichkeiten.

## 🌐 Über das Projekt

Die Website wurde entwickelt, um Atesh Grill Bremen online darzustellen und unseren Kunden eine moderne und übersichtliche Erfahrung zu bieten.

Funktionen der Website:

- 📱 Responsive Design für Handy, Tablet und Desktop
- 🍽️ Speisekarte mit unseren Angeboten
- 📍 Informationen zum Restaurant
- 📞 Kontaktmöglichkeiten
- 🖼️ Präsentation unseres Restaurants und unserer Speisen

## 🛠️ Verwendete Technologien

Dieses Projekt wurde erstellt mit:

- HTML5
- CSS3
- JavaScript
- Astro
